Проект: Самодельный пьезодатчик касания для Sprite экструдера (Ender 3 S1/Klipper)
Версия: Финальная (08.03.2026)
Цель: Автоматическая калибровка Z-оффсета касанием сопла (аналог Bambu Lab) с использованием двух пьезодисков, LM358 и Arduino Pro Mini.

1. 🎯 Концепция и принцип работы
Два пьезодиска устанавливаются симметрично по бокам от канала подачи пластика, между радиатором и пластиковым корпусом экструдера.

При касании соплом стола усилие передается на радиатор, который сжимает диски.

Диски соединены последовательно для увеличения выходного напряжения.

Сигнал усиливается операционным усилителем LM358 и подается на аналоговый вход Arduino Pro Mini (5V).

При превышении порога Arduino замыкает оптрон EL817, который сигнализирует Klipper о срабатывании концевика Z через штатный разъем BLTouch.

3. 🔌 Электрическая схема (текстовое описание)
3.1. Соединение пьезодисков (последовательно)
text
Пьезодиск 1 (+) ─────┬─────── Пьезодиск 2 (–)
Пьезодиск 1 (–) ──┐  └─────── Пьезодиск 2 (+)
                  │
                   └─────── R1 (1 МОм) ────► к пину 2 LM358
Свободные выводы: от Пьезодиск 1 (–) и Пьезодиск 2 (+) идут к схеме.

Важно соблюсти полярность, чтобы сигналы складывались.

3.2. Схема на LM358 (U1)
text
        ┌─────────────────────────────────┐
        │                                 │
        │  R1 (1 МОм) ────────┬─────── Пин 2 LM358 (IN1-)
        │                      │
        │                      ├─── R2 (10 МОм) ──── GND
        │                      │
        │                      ├─── R3 (100 кОм) ──── Пин 3 LM358 (IN1+)
        │                      │
        │  Пин 3 LM358 (IN1+) ─┘
        │
        │  Пин 1 LM358 (OUT1) ──────────────────────► A0 Arduino
        │
        │  Пин 8 LM358 (VCC) ────┬───► +5V
        │                         └─── C1 (0.1 мкФ) ──── GND
        │
        │  Пин 4 LM358 (GND) ────────────────────────► GND
        └─────────────────────────────────┘
3.3. Подключение к Arduino Pro Mini
A0 ← выход LM358 (пин 1)

D9 → R4 (1 кОм) → анод оптрона (пин 1 EL817)

VCC (+5V) ← от разъема BLTouch (через общую шину питания)

GND ← общая земля (от разъема BLTouch)

3.4. Оптопара EL817 (U2)
text
        ┌─────────────────────────────┐
        │  Пин 1 (анод) ← R4 ← D9     │
        │  Пин 2 (катод) ────────────► GND
        │  Пин 3 (эмиттер) ──────────► GND
        │  Пин 4 (коллектор) ─────────► Сигнал (пин 5 разъема BLTouch)
        └─────────────────────────────┘
3.5. Подключение к разъему BLTouch (на Sprite)
Распиновка разъема (проверить мультиметром!):

Пин	Цвет (условно)	Назначение	Куда подключать
1	Белый/Синий	GND	Общая GND схемы
2	Черный/Красный	+5V	Питание (+5V) Arduino и LM358
3	Желтый	Servo	НЕ ИСПОЛЬЗУЕТСЯ
4	Зеленый	GND (дубль)	Можно соединить с GND для надежности
5	Синий/Белый	Signal	Коллектор оптрона (пин 4 EL817)
4. 🛠️ Механическая сборка (ключевые моменты)
4.1. Подготовка дисков
Припаять провода быстро, минимальным количеством припоя.

Заизолировать всю поверхность диска, кроме мест контакта, каптоновой лентой или слюдой.

4.2. Установка в экструдер
Диски размещаются по бокам от центральной трубки подачи пластика, в зазоре между радиатором и пластиковым корпусом.

Под каждый диск с обеих сторон подложить тонкий изолятор.

При установке радиатора убедиться, что диски не перекошены и плотно прилегают.

4.3. Регулировка предварительного натяга
Винты, стягивающие радиатор и корпус, являются регуляторами усилия.

Метод настройки:

Подключить схему и выводить значение АЦП в монитор порта.
Медленно затягивать винты, наблюдая за показаниями.
Идеал: Базовое значение немного выросло, при легком постукивании по соплу – четкие всплески.
Зафиксировать винты каплей лака или фиксатора резьбы.

5. 💻 Программная часть
5.1. Скетч для Arduino Pro Mini
cpp
const int piezoPin = A0;      // Сигнал с LM358
const int optoPin = 9;        // Управление оптроном
const int threshold = 100;     // ПОРОГ (подбирается экспериментально!)
const int delayTime = 50;      // Длительность импульса

void setup() {
  pinMode(optoPin, OUTPUT);
  digitalWrite(optoPin, LOW);
  // Для отладки можно включить Serial.begin(9600);
}

void loop() {
  int val = analogRead(piezoPin);
  
  // Для отладки можно выводить val в Serial
  
  if (val > threshold) {
    digitalWrite(optoPin, HIGH);  // Замыкаем оптрон
    delay(delayTime);
    digitalWrite(optoPin, LOW);
    
    // Небольшая задержка для антидребезга
    delay(150);
  }
  
  delay(10); // Небольшая задержка для стабильности
}
5.2. Подбор порога (threshold)
Загрузить скетч, открыть монитор порта.

Смотреть значения в спокойном состоянии (холостой шум).

Постучать по соплу – увидеть всплески.

Установить порог чуть выше максимального шума, но ниже минимального всплеска при касании.

5.3. Настройка Klipper (printer.cfg)
ini
[stepper_z]
endstop_pin: probe:z_virtual_endstop   # Виртуальный концевик
# position_endstop: 0                   # Закомментировать или удалить

[probe]
pin: ^PB1                               # Пин сигнала от оптрона (УТОЧНИТЬ ПО СВОЕЙ ПЛАТЕ!)
x_offset: 0                              # Так как щупает само сопло
y_offset: 0
z_offset: 0                               # Будет откалибровано позже
speed: 5.0
samples: 3
samples_tolerance: 0.015
samples_tolerance_retries: 3
Важно: Пин ^PB1 приведен для примера (для плат Creality 4.2.2/4.2.7). Уточни свой пин по документации или схеме.

6. ⚠️ Критические предупреждения
Полярность пьезодисков: При последовательном соединении обязательно проверить, что сигналы складываются. Если при постукивании сигнал стал слабее – поменять местами выводы одного из дисков.

Изоляция: Пьезодиск не должен касаться металла радиатора или корпуса ни одной из своих сторон. Это вызовет короткое замыкание сигнала.

Пайка: Перегрев убивает пьезокерамику. Паять быстро, использовать минимальное количество припоя.

Проверка питания: Прежде чем подключать схему к разъему BLTouch, мультиметром убедись, что на пинах 2 (+5V) и 1 (GND) есть нужное напряжение и правильная полярность. Цвета проводов могут быть нестандартными.

Натяг: Не перетяни винты! Это разрушит диски. Настройка ведется плавно, с контролем сигнала.

7. 📈 Ожидаемый результат
При команде PROBE в Klipper сопло будет опускаться, касаться стола, и принтер мгновенно получит сигнал.

Точность срабатывания – в пределах 5-10 микрон (зависит от механики).

Возможность полной автоматической калибровки первого слоя.
